# ARIA v4.0 - 花莲灾害可达性评估系统

## 项目概述

本项目构建了一个集成自动灾害可达性评估系统，结合路网、降雨和地形数据进行综合分析。

## 数据来源

- **路网数据**: OpenStreetMap (通过OSMnx获取)
- **降雨数据**: 第6周克里金插值结果
- **地形数据**: 第4周地形风险类型
- **避灾点数据**: 第3周避灾点位置设置及河流距离数据

## AI诊断日志

### 1. OSMnx数据提取
**遇到的问题**: GeoPandas与OSMnx版本兼容性问题，导致`AttributeError: 'GeoDataFrame' object has no attribute 'active_geometry_name'`
**解决方案**: 
- 降级GeoPandas到0.12.2版本
- 降级OSMnx到1.5.1版本
- 实现手动投影功能绕过内置函数错误

### 2. 等时线多边形生成
**遇到的问题**: 边数据迭代时出现`ValueError: not enough values to unpack (expected 3, got 2)`
**解决方案**:
- 正确处理MultiIndex结构 (u, v, key)
- 修复节点数据重复参数冲突
- 实现稳健的凸包生成算法

### 3. 投影坐标系问题
**遇到的问题**: EPSG:3826投影失败，影响距离计算准确性
**解决方案**:
- 手动实现图投影功能
- 使用GeoPandas的to_crs方法处理节点和边
- 重建图结构保持网络完整性
