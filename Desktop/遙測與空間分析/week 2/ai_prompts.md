# AI 提示紀錄與效果分析

## Prompt 1：基礎資料品質檢查

**使用提示**：
```
請檢查此 CSV 是否有缺值、異常值、重複資料。
```

**AI 回應摘要**：
- 成功發現空值問題：村里缺失209筆、地址缺失3筆、災害類別缺失184筆
- 識別出經度為0的明顯異常值3筆
- 發現305筆重複座標記錄

**效果評估**：
- **優點**：快速識別基礎問題，覆蓋面廣
- **限制**：未深入分析空間合理性，缺乏地理情境判斷
- **改進空間**：可增加台灣地理範圍檢查

## Prompt 2：電話格式標準化

**使用提示**：
```
請幫我修復電話格式不一致問題。
```

**AI 回應摘要**：
- 提供正規表達式清理程式碼
- 處理科學記號轉換 (4.72E+11 → 472000000000)
- 統一分機符號格式 (02-29112281#1209 → 0229112281)

**效果評估**：
- **優點**：提供具體可執行程式碼，處理效果明顯
- **限制**：未考慮台灣電話號碼的區域特性
- **改進空間**：可加入區域碼驗證機制

## Prompt 3：座標合理性驗證

**使用提示**：
```
請檢查經緯度是否在台灣合理範圍內。
```

**AI 回應摘要**：
- 提供台灣座標範圍篩選方式 (經度119-123，緯度21-26)
- 識別超出範圍的異常座標
- 建議精度不足的處理方式

**效果評估**：
- **優點**：準確識別地理範圍問題，提供明確標準
- **限制**：未考慮特殊地區(金門、馬祖)的座標範圍
- **改進空間**：應包含離島地區的座標驗證

## 整體AI表現分析

### AI 優勢
1. **問題識別準確**：能快速發現明顯的數據品質問題
2. **提供解決方案**：給出具體的處理建議和程式碼
3. **處理效率高**：大幅減少人工檢查時間

### AI 局限
1. **地理情境理解不足**：
   - 不知道「營區位置恕不敘明」屬於軍事設施特殊表述
   - 無法判斷大型公園收容人數的合理性
   - 對台灣行政區劃複雜性理解有限

2. **空間分析深度不夠**：
   - 缺乏對座標精度的深入分析
   - 無法識別潛在的系統性座標偏移
   - 對空間分布模式分析有限

3. **政策背景判斷缺失**：
   - 不了解避難所設置的政策背景
   - 無法識別特殊類型避難所的合理性

### 提示優化經驗

**原始提示問題**：
- 過於籠統：「幫我檢查資料問題」
- 缺乏情境：未提供台灣地理背景
- 範圍不明：未指定檢查的重點

**優化後提示**：
```
請從空間座標合理性、行政區完整性、字元編碼、格式一致性四個面向檢查台灣避難收容所資料，特別注意：
1. 台灣本島座標範圍(經度119-123，緯度21-26)
2. 離島地區特殊座標範圍
3. 軍事設施可能的特殊地址表述
4. 中文字元編碼問題
```

**優化效果**：
- 檢查更全面，遺漏問題減少
- 結果更具體，便於後續處理
- 提升AI分析的準確性

## 結論與建議

### 主要發現
1. **AI適合基礎資料清理**：在識別明顯問題方面表現優異
2. **空間分析需人工輔助**：複雜的地理判斷仍需專家知識
3. **提示設計關鍵重要**：具體、情境化的提示能顯著提升效果

### 未來改進方向
1. **建立地理知識庫**：為AI提供台灣特定的地理資訊
2. **開發專用提示模板**：針對空間數據設計標準化提示
3. **人機協作模式**：AI處理基礎問題，人工專注複雜判斷
